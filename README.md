# Reto-3-Grupo-J
Aplicación web de gestión de inventario para una ferretería, desarrollada con React y Supabase. Incluye control de stock en tiempo real, registro de ventas, alertas automáticas de productos mínimos, autenticación de usuarios y dashboard con métricas clave. Enfocada en resolver problemas reales de MIPYMES en República Dominicana.
